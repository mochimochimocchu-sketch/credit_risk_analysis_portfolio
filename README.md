# Credit Risk Analysis Portfolio

## Background
審査業務では判断基準が感覚に依存している部分があり、
適切な審査項目の優先順位、見るべき箇所を特定できれば審査時間短縮につながると考えた。
そのため延滞率に影響する要因を、データ分析で可視化することを目的とした。

## Data
仮想データ5000件を作成し、以下の項目を使用。
- 年齢
- 年収
- 勤続年数
- 他社借入
- 他社延滞回数
- 自社延滞履歴

## Analysis
- SQLによる項目別延滞率の集計
- Pythonによるデータ可視化
- ロジスティック回帰モデルによる延滞予測

## Evaluation
モデル評価として以下を確認。
- Confusion Matrix
- ROC Curve
- AUC

## Result
延滞履歴などの項目が延滞確率に影響する可能性が示された。
審査業務において優先的に確認すべき項目を
定量的に把握する手法として活用できる可能性がある。
